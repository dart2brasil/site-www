shortTitle: dart2js

---

:::version-note
Dart 2.18 remove a ferramenta de linha de comando `dart2js` do pacote Dart,
mas mantém o compilador dart2js.

Use [`dart compile js`](/tools/dart-compile#js) para compilar código Dart
para JavaScript implantável.
:::
