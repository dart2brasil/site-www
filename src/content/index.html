Uma linguagem acessível, portátil e produtiva para aplicativos de alta qualidade em qualquer plataforma

Dart é gratuito e de código aberto  

Acessível
Desenvolva com uma linguagem de programação consistente, concisa e fortemente tipada, que oferece recursos modernos como segurança contra nulidade e correspondência de padrões.
Desenvolvimento
produtivo
Faça alterações iterativamente: use o hot reload para ver o resultado instantaneamente no seu aplicativo em execução.
Portátil e rápido
em todas as plataformas
Compile para código de máquina ARM, x64 e RISC-V para dispositivos móveis, desktop e backend. Ou, compile para JavaScript e WebAssembly para a web.
Acessível
  • Async-await maduro e completo para interfaces de usuário contendo código orientado a eventos, combinado com concorrência baseada em isolates.
  • Escreva código seguro e conciso usando recursos como null safety seguro, if de coleção, e pattern matching.
  • Uma linguagem de programação consistente, com uma sintaxe fácil de aprender e familiar.
Desenvolvimento Produtivo
  • Faça mudanças no seu código fonte de forma iterativa, usando hot reload para ver instantaneamente o efeito no aplicativo em execução.
  • Escreva código usando um sistema de tipos flexível com análise estática rica e poderosas ferramentas configuráveis.
  • Faça perfilagem, log, e depuração com o editor de código de sua escolha.
Rápido em todas as plataformas
  • Compile apps para código de máquina nativo usando AOT para início instantâneo.
  • Alvo para a web com compiladores completos, maduros e rápidos para JavaScript & WebAssembly.
  • Execute código de backend que dá suporte ao seu app, escrito usando uma única linguagem de programação.
