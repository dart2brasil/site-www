Uma linguagem acessível, portátil e produtiva
para aplicações de alta qualidade em qualquer plataforma

Dart é gratuito e de código aberto  

Acessível
Desenvolva com uma linguagem de programação consistente, concisa e fortemente tipada que oferece recursos modernos como null safety e pattern matching.
Portátil e rápido
Compile para código de máquina ARM, x64 e RISC-V para mobile, desktop e backend. Ou compile para JavaScript e WebAssembly para a web.
Produtivo
Faça alterações iterativamente: use hot reload para ver o resultado instantaneamente em sua aplicação em execução.
Acessível
  • Async-await maduro e completo para interfaces de usuário contendo código orientado a eventos, pareado com concorrência baseada em isolate.
  • Escreva código seguro e conciso usando recursos como null safety sólido, collection if e pattern matching.
  • Uma linguagem de programação consistente, com uma sintaxe familiar e fácil de aprender.
Portátil e rápido
  • Compile AOT aplicações para código de máquina nativo para inicialização instantânea.
  • Desenvolva para a web com compiladores completos, maduros e rápidos para JavaScript e WebAssembly.
  • Execute código backend para suportar sua aplicação, escrito usando uma única linguagem de programação.
Produtivo
  • Faça alterações no seu código-fonte iterativamente, usando hot reload para ver o efeito instantaneamente na aplicação em execução.
  • Escreva código usando um sistema de tipos flexível com análise estática rica e ferramentas poderosas e configuráveis.
  • Faça profiling, logging e debugging com o editor de código de sua escolha.
