Uma linguagem acessível, portátil e produtiva para aplicativos de alta qualidade em qualquer plataforma

Dart é gratuito e de código aberto  

Acessível
Desenvolva com uma linguagem de programação consistente, concisa e fortemente tipada, que oferece recursos modernos como null safety (segurança nula) e pattern matching (correspondência de padrões).
Desenvolvimento Produtivo
Faça alterações iterativamente: use o hot reload (recarregamento instantâneo) para ver o resultado instantaneamente em seu aplicativo em execução.
Portátil e rápido
em todas as plataformas
Compile para código de máquina ARM, x64 e RISC-V para mobile, desktop e backend. Ou, compile para JavaScript e WebAssembly para a web.
Acessível
  • Async-await maduro e completo para interfaces de usuário contendo código orientado a eventos, combinado com concorrência baseada em isolates.
  • Escreva código seguro e conciso usando recursos como null safety (segurança nula), collection if, e pattern matching (correspondência de padrões).
  • Uma linguagem de programação consistente, com uma sintaxe familiar e fácil de aprender.
Desenvolvimento Produtivo
  • Faça alterações no seu código fonte iterativamente, usando o hot reload (recarregamento instantâneo) para ver o efeito instantaneamente no aplicativo em execução.
  • Escreva código usando um sistema de tipos flexível com análise estática rica e ferramentas configuráveis poderosas.
  • Faça profiling, logging, e debugging com o editor de código de sua preferência.
Rápido em todas as plataformas
  • Compile AOT (Ahead-of-Time) aplicativos para código de máquina nativo para inicialização instantânea.
  • Tenha como alvo a web com compiladores completos, maduros e rápidos para JavaScript e WebAssembly.
  • Execute código backend que dá suporte ao seu aplicativo, escrito usando uma única linguagem de programação.
